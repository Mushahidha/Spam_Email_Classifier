# 📧 Spam Email Classifier

A Machine Learning project that classifies emails as Spam or Ham using Python, Streamlit, and Scikit-learn.

## Features
- Detects Spam and Ham emails
- Built with Naive Bayes
- Interactive Streamlit interface
- Simple and beginner-friendly

## Technologies Used
- Python
- Streamlit
- Pandas
- Scikit-learn

## Run the Project

```bash
pip install -r requirements.txt
streamlit run app.py
```
